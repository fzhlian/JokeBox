# JokeBox
🎭 笑话盒（JokeBox）

一款支持分龄过滤、智能去重、语音朗读、多来源管理的高质量笑话 App。
让笑声更安全、更纯净、更智能。

🧠 产品理念

在信息过载的时代，真正有质量、适合不同年龄段的内容越来越难筛选。
笑话盒（JokeBox） 致力于：

让不同年龄段用户看到真正适合自己的内容

通过智能去重技术避免“重复段子刷屏”

支持多语言与多来源

提供自然语音朗读，随时随地轻松收听

👶👦🧑 分龄安全机制

首次启动必须选择年龄段：

👶 少年模式（最严格过滤）

👦 青年模式（适度开放）

🧑 成人模式（更广内容范围）

选择后将被锁定（除非清除应用数据），确保内容环境稳定、安全。

系统采用：

本地分级策略

敏感内容过滤

多层内容审核规则

为不同年龄层提供更合适的阅读体验。
🔄 智能去重系统

告别重复段子！

笑话盒采用：

文本标准化处理

精确哈希去重

SimHash 相似度识别（相近内容自动过滤）

同语言 + 同年龄段范围比对

真正做到：
不重复、不刷屏、只留优质内容。

⚙ 多来源内容支持
✅ 内置合规内容源

预设授权或公开许可的数据源

清晰展示来源与许可说明

➕ 用户自带来源（可选）

支持添加 API / RSS 源

支持导入本地文件（JSON / CSV / TXT / HTML）

启用前系统会提示合规声明，提醒用户确认内容来源合法合规。

🚀 异步后台处理

抓到多少，处理多少

处理完成即可播放

后台自动去重、过滤、分类

不阻塞界面操作

始终流畅体验。

🔊 自然语音朗读

支持：

多种音色（温柔 / 活泼 / 稳重等）

语速与音调调节

本地 TTS + 可选云端增强语音

解放双眼，让笑声陪伴你。

🌍 多语言支持

界面语言默认跟随系统

内容语言可自动匹配或手动指定

支持简体中文 / 繁体中文 / English 等

真正国际化体验。

⭐ 个性化功能

上一个 / 下一个浏览

已播内容不重复

星标收藏喜欢的笑话

自动更新内容库

播完自动提醒更新或重置

🔐 隐私与合规

所有处理优先在本地完成

用户自定义来源需主动确认合规声明

不内置任何绕过限制的抓取功能

用户可随时清空自定义来源与数据

📱 适用平台

Android

HarmonyOS

HarmonyOS NEXT

统一体验，多端同步架构设计。

想给孩子一个安全笑话环境的家长

希望避免低质重复段子的用户

喜欢语音收听内容的人

想自定义内容来源的进阶用户
