JokeBox Harmony Source Package v0.1.1

Contents
- harmony/: HarmonyOS Stage project
- harmony-next/: HarmonyOS NEXT Stage project

Build in DevEco Studio
1. Open either harmony/ or harmony-next/ as project root.
2. Sync dependencies.
3. Build HAP: Build > Build Hap(s)/APP(s) > Build Hap(s).
4. Build APP if required in the same menu.

Note
- This package is source-only and requires DevEco Studio toolchain for binary build.
